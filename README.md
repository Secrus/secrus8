# secrus8 - a CHIP-8 iterpreter

This is a simple (WIP!) [CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) interpreter. 

## Running

You can try the interpreter on a test ROM in this repo.

```
secrus8 ibm-logo.ch8
```
